# 💳 Credit Card Default Prediction

Predict whether a customer will default on 
their credit card payment next month.

## 📊 Dataset
- Source: UCI / Kaggle
- 30,000 customer records

## 🤖 Models Used
- Logistic Regression
- Random Forest
- XGBoost

## ⚙️ How to Run

### Install libraries:
pip install -r requirements.txt

### Run the code:
python credit_card_user_input.py

## 📈 Results
| Model               | ROC-AUC |
|---------------------|---------|
| Logistic Regression | 0.72    |
| Random Forest       | 0.76    |
| XGBoost             | 0.78 🏆 |
